# Tooling Instructions

These instructions apply under `tools/`.

- Do not install tools globally.
- Do not add unapproved network access.
- Do not print secrets or full sensitive payloads.
- Keep scripts deterministic unless the command name and docs clearly mark them informational.
- Write artifacts only to controlled ignored directories.
- Keep scripts cross-platform where practical.
- Fail honestly when an implementation is phase-gated or unavailable.
- Do not publish, deploy, push, tag, release, or mutate GitHub settings.
