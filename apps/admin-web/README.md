# @nelyohealth/admin-web

Next.js 16 shell workspace for the admin web application foundation.

## Public API

- Exports `adminWebApplicationBoundary` and shell descriptors for P02-ISS-012 topology/runtime validation.
- Provides a synthetic-only shell route for foundation verification.
- Wires typed API client construction through `@nelyohealth/api-client`.

Operational admin product workflows remain out of scope in this package.
