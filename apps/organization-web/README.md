# @nelyohealth/organization-web

Next.js 16 shell workspace for the organization web application foundation.

## Public API

- Exports `organizationWebApplicationBoundary` and shell descriptors for P02-ISS-012 topology/runtime validation.
- Provides a synthetic-only shell route for foundation verification.
- Wires typed API client construction through `@nelyohealth/api-client`.

Benefits and claims product features remain out of scope in this package.
