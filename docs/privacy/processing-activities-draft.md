|---|---|---|---|---|
| Clinical care|Not automatically|Clinical care, legal obligation, contract, vital interest, or other reviewed basis question|Legal Counsel + Clinical Lead|REQUIRES_APPROVAL |

| Marketing|Yes if used|Consent or approved marketing basis question|Legal Counsel + Privacy Counsel|REQUIRES_APPROVAL |
| Optional analytics|Possible optional consent|Legitimate-interest or consent question; no decision in P00-11|Legal Counsel + Privacy Counsel|REQUIRES_APPROVAL |
| Emergency processing|Not relied on as blanket consent|Vital-interest/emergency/public-interest question for P00-12|Legal Counsel + Clinical Lead|REQUIRES_APPROVAL |
| Security monitoring|Not generally consent|Security obligation/legitimate-interest question|Legal Counsel + Security Lead|REQUIRES_APPROVAL |
| Guardian/delegation|Consent and authority distinct|Authority evidence and lawful basis question|Legal Counsel + Privacy Counsel|REQUIRES_APPROVAL |
