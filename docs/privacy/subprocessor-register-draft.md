## Due-Diligence Gate

A subprocessor must not be approved until initial assessment, security questionnaire, privacy assessment, legal assessment, data-flow verification, contract, test, pilot approval, production approval, periodic review, material-change review, termination, and data return or deletion are complete.

## Material Change

Material-change triggers include new country, new subprocessor, new data category, new purpose, new retention, security incident, contract change, acquisition, service architecture change, new AI or model use, or session replay activation.
