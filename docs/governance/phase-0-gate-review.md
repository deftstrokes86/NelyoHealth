## Phase 1 Foundation Scope

Permitted: repository foundation, monorepo setup, TypeScript/package-manager baseline, CI gates, static analysis, synthetic-data tooling, local/dev/staging skeletons, secrets-management boundary, modular-monolith skeleton, design-token implementation, content-schema implementation, Motion governed primitives, UI UX Pro Max reviewed project-local installation, project-scoped Codex browser setup, Playwright Test setup, browser smoke page, synthetic auth fixtures, initial application shells, observability skeleton, feature flags, and documentation automation.

Prohibited: production launch, real patient/provider data, live pharmacy/lab/clinical/payment/HMO/employer operations, unapproved clinical content, unapproved legal/regulatory claims, and blocked capability implementation beyond interfaces or abstractions.
