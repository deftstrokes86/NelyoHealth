## Review Rubric

Hierarchy, alignment, spacing, typography, color, contrast, density, component consistency, navigation, task clarity, CTA clarity, state design, error recovery, responsive behavior, accessibility, motion, reduced motion, content alignment, role appropriateness, privacy, clinical safety, and performance must be classified. A numerical score alone is not approval.

## Screenshot Matrix

Plan screenshots for desktop, tablet, mobile, loading, empty, populated, error, offline, unauthorized, payment pending, payment failed, provider detail locked, provider detail authorized, urgent, emergency, and reduced motion. All evidence uses synthetic data. Blind snapshot-baseline updates are prohibited.
