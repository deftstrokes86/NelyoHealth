# NelyoHealth Content Guide

## Positioning

NelyoHealth is coordinated healthcare infrastructure for Nigeria,
connecting patients, clinicians, pharmacies, labs, hospitals, employers,
HMOs, care partners, guardians, families and diaspora sponsors.

## Universal Rules

-   One audience per page.
-   One goal and one primary CTA.
-   Structure: Hero → Problem → Solution → Benefits → How it Works →
    Trust → FAQ → CTA.
-   Lead with outcomes, not features.
-   Explain workflows, not technology.
-   Never invent facts, testimonials or metrics.
-   Use plain English.
-   AI assists; clinicians decide.
-   End every page with a CTA.

## Validation

Check audience, goal, CTA, supported claims, workflow clarity and next
step before publishing.
